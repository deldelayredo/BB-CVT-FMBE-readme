# BB 转 FMBE 命令转换器

一个静态 HTML 工具，用于将 Blockbench 导出的 Minecraft 基岩版模型 JSON 转换为 FMBE 兼容的命令文本。

## 功能

- 将基岩版模型中的骨骼和方块转换为 FMBE 命令
- 支持多种 FMBE 版本选择
- 处理方块旋转、枢轴点和骨骼层级关系
- 对包含父子骨骼结构的模型支持动态转换
- 支持自定义前缀/后缀输出，并可插入方块编号
- 支持上传 `.json` 模型文件

## 使用方法

1. 在浏览器中打开 `index.html`。
2. 设置换算系数 `k`（默认 `1`）。
3. 选择所需的版本。
4. 将 Blockbench 导出的基岩版模型 JSON 粘贴到输入框，或上传 `.json` 文件。
5. 可选地填写自定义前缀或后缀，使用 `<>` 占位符插入方块编号。
6. 点击 `转换`。
7. 使用单个复制按钮或 `全部复制` 按钮复制生成的输出命令。

## 说明

- 如果模型包含骨骼层级关系，应用会提示是否启用动态骨骼转换。
- 该转换器针对 Blockbench 导出的基岩版模型 JSON 设计，可能不支持其他格式。
- 自定义前缀/后缀中的内容会进行安全处理。
